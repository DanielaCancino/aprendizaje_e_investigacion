# Sobre este archivo

## Aprendizaje e investigación

En este proyecto se analiza un conjunto de datos para examinar el impacto que ha tenido el trabajo flexible —como el teletrabajo y el modelo híbrido— en la salud mental de los empleados.  

El objetivo de esta investigación es responder preguntas específicas sobre cómo cada modalidad laboral influye en la salud mental según distintos factores. Se busca identificar patrones asociados al estrés, aislamiento social y satisfacción laboral, proporcionando a las organizaciones una base sólida para diseñar políticas de trabajo remoto más saludables y estrategias efectivas de apoyo al bienestar mental de sus equipos.  

## Características clave

- **Encuestados diversos:** Datos de empleados pertenecientes a diferentes edades, profesiones, e industrias.  
- **Perspectiva sobre salud mental:** Explora los efectos del trabajo remoto, híbrido y presencial en distintas dimensiones del bienestar.  
- **Datos analizados:** El conjunto incluye variables como edad, género, rol laboral, industria, tipo de modalidad de trabajo, nivel de estrés reportado, diagnósticos de salud mental, sensación de aislamiento social y satisfacción con el trabajo.
